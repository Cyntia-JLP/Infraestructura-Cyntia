# Infraestructura-Cyntia

En este repositorio se recopilan de forma estructurada todas las guías paso a paso correspondientes a cada una de las fases del proyecto relacionadas con la infraestructura del mismo. 

Además de la documentación, también se incluyen los playbooks, configuraciones y fragmentos de código relevantes que forman parte del desarrollo e implementación del proyecto, permitiendo así reproducir y mantener la infraestructura de manera eficiente.

Para consultar la documentación completa, organizada y redactada en mayor profundidad, podéis acceder a nuestro GitBook en el siguiente enlace: 

    https://cyntia.gitbook.io/cyntia/cyntia-proyecto-asir/infraestructura-cyntia
